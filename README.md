#  **InfoCar Pro** - Sistema Oficina Mecânica COMPLETO

| Módulo | Features |
|--------|----------|
| **Dashboard** | Métricas reais, gráficos, OS recentes |
| **Ordens Serviço** | 30+ serviços, marcas/modelos, campos dinâmicos |
| **Estoque** | Alertas críticos, busca, histórico |
| **Clientes** | WhatsApp direto, histórico OS, placas múltiplas |
| **Relatórios** | Gráficos Chart.js, filtros data, export PDF/Excel |
